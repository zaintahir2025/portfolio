AI & Services Mobile · Web · Desktop

Ustaad — Agentic Service Platform

A production-minded service-booking app for Pakistan's informal economy. Ustaad uses natural language processing to understand service requests in English, Urdu, and Roman Urdu, then intelligently ranks nearby providers using a custom multi-factor scoring algorithm.

🤖 AI-Powered NLP Parser

Custom multilingual parser extracts service type, location, time, language, urgency, and confidence from natural language input — no LLM API key required.

📊 Multi-Factor Ranking Engine

Provider ranking weighs distance (haversine), availability, rating, reliability, completed jobs, urgency, and price to surface the best match.

🗺️ Free Map Stack

OpenStreetMap tiles via flutter_map, Nominatim for place search, OSRM for route/ETA estimates — with offline Islamabad sector fallbacks.

🔐 Supabase Backend

Supabase Auth with email confirmation, PostgreSQL with RLS policies, Storage for avatars, and structured workflow trace event logging.

🎤 Voice Input

Device speech recognition via speech_to_text — users can describe their service needs by voice in any supported language.

📱 Provider Architecture

Flutter Provider pattern with orchestration state, offline catalog fallback, booking lifecycle management, and saved professionals.

Social & Real-Time Mobile · Web · Desktop

Voxora — Social Voice Chat Network

An advanced social networking application merging modern social feeds, low-latency peer-to-peer WebRTC voice calling, and AI-driven multiplayer games into a single, cohesive flat-design UI — built entirely with Flutter and Supabase.

📡 WebRTC Peer-to-Peer Calling

Decentralized voice calling via flutter_webrtc with SDP offer/answer exchange, ICE candidate negotiation, and NAT traversal — using Supabase Realtime as the signaling server.

🧠 Minimax AI with Alpha-Beta Pruning

Chess and game AI opponents using the Minimax algorithm with heuristic board evaluation and alpha-beta pruning for optimal move selection at depth 3–4.

🏗️ MVCS Architecture

Model-View-Controller-Service pattern with Provider-based reactive state management. AppProvider for global state, BotGameProvider for game state machines.

🎨 Engineered Design System

Material 3 dynamic color schemes with elegant deep navy dark mode (#0B192C) and ultra-minimal light mode. Zero shadows — flat 1px borders for modern aesthetics.

📱 Responsive Breakpoints

Desktop sidebar layout vs. mobile BottomNavigationBar with MediaQuery breakpoint at 980px. Dynamic avatar fallbacks with graceful error handling.

🚀 CI/CD Pipeline

GitHub Actions workflow: flutter analyze as code quality gate → flutter build web (WASM) → deploy to GitHub Pages automatically on every push.
